# WritingToolLight (a LibreOffice Extension based on LanguageTool) 

***NOTE: WritingTool requires Java 17 or higher now***

WritingToolLight is an extension for OpenOffice/LibreOffice that enhances text editing with grammar checking (and additional spell checking) based on LanguageTool.
The following features are offered:
* The application runs locally on the computer by default. Therefore, no remote server is required.
* The use of a remote server, including the LanguageTool Premium service, is still supported.
* In addition to the spell checker of OpenOffice/LibreOffice, the LanguageTool spell checker is used.
* A configuration dialog allows users to change settings such as:
  * Easily enable/disable optional rules
  * Easily disable/re-enable default rules
  * Define custom colors and styles for rule groups or individual rules
  * Change parameters for specific special rules
  * Define profiles for checking different document types


You can find more information (requirements, licence, downloads, etc.) here: https://writingtool.org
